🔧 Chatbot Configuration Test

This page helps you diagnose chatbot configuration issues.

📋 Configuration Steps:

For Vercel Production:

  1. Go to your Vercel Dashboard
  2. Select your project
  3. Go to Settings → Environment Variables
  4. Add: OPENAI_API_KEY = your-api-key
  5. Select all environments (Production, Preview, Development)
  6. Save and redeploy

For Local Development:

  1. Open chatbot-config.js
  2. Replace sk-your-actual-openai-api-key-here with your real API key
  3. Save the file
